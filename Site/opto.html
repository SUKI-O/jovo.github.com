We have developed a family of tools for the purposes of inferring precise spike trains from noisy calcium imaging data, and then using these spike times to infer connectivity.  These tools have been tested both in vivo and in vitro, using organic dyes and genetic sensors, with various preps, cell types, imaging rates, and labs around the world.


Publications:


  1. 1)Fast non-negative deconvolution for spike train inference from population calcium imaging. Vogelstein JT, Packer AM, Machado TA, Sippy T, Babadi B, Yuste R, Paninski L. J Neurophysiology, in press.  Abstract.   PDF.  Code.  Repository (containing archives of all versions of data, figures, code, and manuscript). 


The above manuscript documents our relatively simple theory and algorithm for inferring spike trains from calcium imaging.  We recommend trying to use this code first.  Please contact me with any difficulties or exciting results!


  1. 2)Spike inference from calcium imaging using sequential Monte Carlo methods. Vogelstein JT, Watson BO, Packer AM, Jedynak B, Yuste R, Paninski L. Biophysical Journal, July 2009.  Abstract.  PDF.  Code.  Repository (containing archives of all versions of data, figures, code, and manuscript).


The above manuscript describes a more complicated nonlinear model accounting for fluorescence fluctuations to due spiking activity.  The associated algorithm has certain desirable features, but lacks robustness relative to the simpler and faster approach.  Recommended only if the fast version is deemed inadequate for some reason.


  1. 3)A Bayesian approach for inferring neuronal connectivity from calcium fluorescent imaging data. Mishchenko Y, Vogelstein JT, Paninski L.  PDF. Repository (containing archives of all versions of data, figures, code, and manuscript).


The above manuscript describes the theoretical limitations of inferring connectivity from calcium imaging data.
